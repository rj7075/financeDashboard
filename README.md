# financeDashboard

## Overview
A responsive finance dashboard to track transactions, visualize spending, and simulate role-based UI.

## Tech Stack
- React
- Tailwind CSS
- Zustand
- Recharts

## Features
- Summary cards (balance, income, expense)
- Charts (trend + category breakdown)
- Transactions table with filtering
- Role-based UI (admin/viewer)
- Insights section
- Dark mode
- Local storage persistence

## Approach
Focused on clean UI, modular structure, and scalable state management using Zustand.

## Setup
npm install
npm run dev
